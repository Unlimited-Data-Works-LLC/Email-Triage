How labels are applied

  • Bulk on triage results — pick a label from the dropdown above the results table, check the messages you want tagged, click "Tag selected".
  • Rule-driven — every label you create on this page appears as a checkbox in the rule editor. Tick it on any rule to attach the label to every message that rule matches, on top of the rule's suggested category. Multiple labels per rule are fine — tick any number, all ticked labels attach at fire time.
  • Where the rule editor lives — My Settings → Rules.
