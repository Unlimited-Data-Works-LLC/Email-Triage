Session timeout

§164.312(a)(2)(iii) Automatic Logoff. Industry norms: EHR / clinical = 10–15 min; HIPAA research portal = 30 min. Hard ceiling of 30 min on the HIPAA cap is enforced in code. Effective TTL = min(operator pick, hipaa cap) when HIPAA mode is on. {% if hipaa_mode %} HIPAA mode is on — non-HIPAA TTL disabled; effective TTL is the HIPAA cap. {% endif %}

Login rate limit

Per-email + per-IP rolling-window counters guarding OTP / WebAuthn / dev-keypair / passkey login surfaces against brute-force and credential-stuffing. Source of truth is the auth_events table; the lockout trips when failure rows in the window exceed the threshold. max=0 disables that scope. §164.312(a)(2)(i) "Unique User Identification."

Recommended: per-email 10 / 600s, per-IP 30 / 600s. Tune per-IP higher (or set 0) when multiple legitimate users share a NAT. Both checks run; first to trip raises a login_lockout audit row.
