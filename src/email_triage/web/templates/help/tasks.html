Help — Common Tasks

This page walks you through common things you'll want to do with Email Triage. Each task has step-by-step directions matching the actual buttons and field labels in the app. New here? Start with task 1 (route mail by sender) — it's the most common.

Setting up a new mailbox? See the Gmail + Office 365 setup guides for step-by-step OAuth + push / poll walkthroughs.

The directions below match the names on the screen. If a button looks greyed out, the page is telling you something — usually that a setting earlier in the page needs to be filled in first, or that the action only applies to a different account type. Hover the little ℹ️ icons for plain-language explanations of any field.

Jump to a task:
  1. Route all mail from one sender to a category
  2. Pause an account while you're on vacation
  3. Let someone else triage your mail (add a delegate)
  4. Set up a daily digest
  5. Turn off AI draft replies for one account
  6. See what training data has been captured
