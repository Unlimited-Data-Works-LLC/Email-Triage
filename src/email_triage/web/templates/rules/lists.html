Lists provide hints to the AI to improve classification accuracy. They are not hard overrides — the AI considers them alongside email content. For an entry to fire, the message must match the list AND the AI must agree the suggested category fits.

Create List

A list groups one or more rules under a suggested category. A new list does nothing by itself — add at least one rule below (sender, sender domain, or subject phrase) so the list can match incoming mail. You can fill in the first rule here, or leave the rule fields blank and add rules later from each list.
